# Contributing to $EVERYTHING

First off, thanks for being part of the $EVERYTHING community! Every contribution matters — whether it's code, memes, or vibes.

## How to Contribute

### Reporting Bugs
- Check existing [Issues](https://github.com/everythingcoin/everything/issues) first
- Use the bug report template
- Include screenshots if it's a visual issue
- Describe steps to reproduce

### Suggesting Features
- Open an issue with the `enhancement` label
- Describe the feature and why it would benefit $EVERYTHING
- Be open to discussion

### Code Contributions

1. **Fork** the repository
2. **Create** a feature branch: `git checkout -b feature/amazing-feature`
3. **Commit** your changes: `git commit -m 'Add amazing feature'`
4. **Push** to the branch: `git push origin feature/amazing-feature`
5. **Open** a Pull Request

### Code Style
- Keep it simple and readable
- Use meaningful variable/function names
- Comment complex logic
- Test your changes locally before submitting

### Meme Contributions
Got a fire meme? We love it. Share it in:
- Our [Telegram](https://t.me/everythingcoin) group
- The `#memes` channel on [Discord](https://discord.gg/everythingcoin)
- An issue with the `meme` label

## Code of Conduct

- Be respectful and inclusive
- No spam or self-promotion
- No financial advice or price manipulation
- Keep discussions constructive
- Have fun — it's a meme coin!

## Recognition

All contributors will be recognized in our community. Outstanding contributions may be highlighted on our social channels.

---

**Remember: $EVERYTHING is a community project. Your contribution, no matter how small, helps make it better for everyone.**
